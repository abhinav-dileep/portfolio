# Portfolio Website Specification

## **Goal**
Build a **three-page portfolio website** using **Next.js** and **TailwindCSS**.

---

## **Theme**
Final Year **BCA Graduate Student** | Aspiring **Frontend Developer**

---

## **Pages & Requirements**

### **1. Home (Index) Page**
- **Hero Section**
  - Title + Short Description
- **About Section**
  - Brief introduction with a round profile pic on right side
- **Education Section**
  - Display education in **chronological order** (latest first)
  - Include **flow-like appearance** for:
    - Degree
    - Plus Two
    - SSLC

---

### **2. Skills Page**
- **Hero Section**
- **Stats Section**
- **Skills Display**
  - Modern, iconic layout
  - **Circular progress rings** for skill levels
  - **Modal popup** for each skill with short description
- **Recent College Projects Section**
  - Add a carousel type infinite scroll with a modal popup description
  - Include 5 projects only
  - Include the techstack used (chips), an image and description on each tile

---

### **3. Contact Page**
- **Hero Section**
  - Title + Short description
- **Contact Form**
  - Include FullName, Email, Mobile and a Message with a gradient send button
  - Model popup which shows a success message after filling and clicking the send button
- **Social Media Links**
  - Call to Action title with a short description
  - GitHub, LinkedIn, Medium (with icons)

---

## **Global Requirements**
- **Common Header & Footer**
- **Responsive Navigation**
  - Hamburger menu for mobile
  - Collapse/expand icon change
  - Sidebar in mobile view
- **Dark/Light Mode Toggle**
  - Proper background color contrast

---

## **Tech Stack**
- **Next.js** (Routing, Components)
- **TailwindCSS** (Styling, Responsive Design)
